If you have Docker and Docker Compose installed, you can get vantagesuite-e up and running in minutes:

```bash
git clone https://github.com/daniellopez882/vantagesuite-e
cd vantagesuite-e
# Follow the deployment instructions in the wiki (coming soon)
```

### Manual Installation

For developers looking to contribute or customize:

1.  Install the [Bench CLI](https://github.com/frappe/bench).
2.  Create a new site: `bench new-site vantage.local`.
3.  Get and install the app:
    ```bash
    bench get-app https://github.com/daniellopez882/vantagesuite-e
    bench --site vantage.local install-app vantagesuite-e
    ```
